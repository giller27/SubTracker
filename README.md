# 📊 SubTrack - Трекер Підписок та Витрат

Простий, потужний i красивий календар для управління всіма твоїми підписками з одного місця.

## 🎯 Основні можливості

### 📈 **Dashboard (Панель управління)**

- 📊 Проглядай загальні витрати на місяць і рік одним поглядом
- 📅 Графік витрат по днях місяця
- 🔔 Сповіщення про наступні платежи
- 📂 Розподіл витрат по категоріях

### 🎬 **Мої підписки**

- 📝 Повний список активних сервісів з детальною інформацією
- 🔍 Пошук по назві сервісу
- 🏷️ Фільтрація по категоріях (Стрімінг, Робота, Хмарні сховища, Спорт)
- 📊 Сортування по ціні, даті додавання або назві
- ✏️ Редагування і видалення підписок

### ➕ **Додати сервіс**

- 🎯 Популярні сервіси (Netflix, Spotify, iCloud та ще 12+)
- 🔎 Пошук в базі популярних сервісів
- ➖ Ручне введення для локальних сервісів
- ⏰ Вибір циклу оплати (щомісячна, щорічна, щотижнева, щоквартальна)
- 📂 Розподіл по категоріях

### 📊 **Аналітика**

- 📈 Графік витрат за останні 6 місяців
- 🥧 Розподіл витрат по категоріях (круговий графік)
- 💡 Поради щодо економії (дублювання, дорогі сервіси, великі витрати)
- 🔮 Фінансовий прогноз на місяці, рік, і наступні 3 місяці

### 📅 **Календар платежів**

- 🗓️ Візуальна сітка місяця з датами платежів
- 📍 Мітки сервісів на конкретних датах
- 🔔 Список усіх платежів з сумами
- ◀️ ▶️ Навігація по місяцях

### ⚙️ **Налаштування**

- 💱 Вибір основної валюти (₴ UAH, $ USD, € EUR, £ GBP)
- 🔔 Налаштування сповіщень (за скільки днів попереджати)
- 📧 Сповіщення на пошту
- 🤖 Telegram бот інтеграція
- 💾 Експорт даних у CSV або JSON
- 📥 Імпорт даних з JSON файлу
- 🧹 Видалення всіх даних

## 🛠️ Технічна частина

### Стек технологій:

- **HTML5** - структура
- **CSS3** - стилізація (з адаптивним дизайном)
- **Vanilla JavaScript** - логіка додатку
- **Chart.js** - графіки і діаграми
- **Day.js** - робота з датами
- **Font Awesome** - іконки
- **LocalStorage** - збереження даних (без сервера)

### Особливості реалізації:

- 📦 Нульові залежності на серверу - все містить в собі
- 💾 Усі дані зберігаються локально в браузері (LocalStorage)
- 🔄 Автоматичні розрахунки цикли платежів
- 📱 Повне адаптивне мобільний дизайн
- ⚡ Швидке завантаження і працює без інтернету

## 🚀 Як користуватися

### 1. Відкрити сайт

Просто відкрий `index.html` у браузері.

### 2. Додати першу підписку

- Перейди на вкладку "➕ Додати"
- Вибери з популярних (Netflix, Spotify і т.д.) або введи свої дані
- Вибери цикл оплати і категорію
- Натисни "Додати підписку"

### 3. Переглянути статистику

- **Dashboard** - швидкий огляд витрат
- **Аналітика** - графіки та поради щодо економії
- **Календар** - коли наступні платежи

### 4. Налаштування

- Вибери свою валюту
- Налаштуй сповіщення на потрібну кількість днів
- Експортуй дані для резервної копії

## 📊 Приклад даних

Коли ти додаси кілька підписок, ти будеш бачити:

- "На цьому місяці: 500 ₴"
- "На цей рік: 6000 ₴"
- Графіки витрат по днях і сумаційні тренди
- Поради: "У вас 4 підписки на стрімінг. Розглянь скасування деяких."

## 🎨 Дизайн

- 🎨 Яскраві, але зручні кольори
- 📱 Адаптивний дизайн для всіх пристроїв
- ✨ Плавні переходи і анімації
- 🌙 Чистий, мінімалістичний інтерфейс

## 🔒 Безпека і приватність

- ✅ Усі дані зберігаються **тільки** на твоєму комп'ютері
- ✅ Ніькуди не передаються на сервер
- ✅ Ти можеш видалити все за один клік

## 📝 Невеликі поради

1. **Регулярно перевіряй** вкладку "Аналітика" для отримання порад щодо економії
2. **Встави сповіщення** на потрібну кількість днів (рекомендуємо 3 дні)
3. **Експортуй дані** як мінімум раз на місяць для резервної копії
4. **Інформуй себе** - знай свій бюджет на місяць!

## 🐛 Проблеми?

Якщо щось не працює:

1. Перезавантаж сторінку (Ctrl+R або Cmd+R)
2. Очисти кеш браузера
3. Переконайся, що браузер підтримує LocalStorage
