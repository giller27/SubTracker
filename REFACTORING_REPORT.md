## 🚀 Як користуватися

### Для редагування сторінки:

**Крок 1**: Визначте яку сторінку хочете редагувати
- Dashboard, Мої підписки, Додати, Аналітика, Календар, Налаштування

**Крок 2**: Знайдіть файли
- HTML: `pages/[назва-сторінки].html`
- JavaScript: `js/pages/[назва-сторінки].js`

**Крок 3**: Редагуйте
- Змінюйте HTML структуру
- Змінюйте функції JavaScript

**Крок 4**: Перевірте
- Оновіть браузер или F5
- Перевірте функціональність

### Приклад редагування Dashboard:

1. Відкрийте `pages/dashboard.html`
2. Змініть структуру карток або додайте новий елемент
3. Відкрийте `js/pages/dashboard.js`
4. Додайте код для нового елемента у функцію `updateDashboard()`
5. Збережіть обидва файли
6. Перезавантажте сторінку

## ⚠️ Важливо!

1. **Порядок завантаження скриптів важливий!**
   - Спочатку загружаються `config.js`, `store.js`, `calculations.js`
   - Потім `navigation.js` і модулі сторінок
   - На останку `init.js`

2. **Глобальні змінні корреспондуються:**
   ```javascript
   store        // Завжди доступна
   calculations // Sempre доступна
   CONFIG       // Завжди доступна
   ```

3. **localStorage формат не змінився**
   - Старі дані будуть автоматично завантажені
   - Сумісна з попередніми версіями
